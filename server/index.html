Todo API Documentation

Users

POST /users

Sign up

- Provide email and password properties on the object. - Password must be at least 6 characters.

Todos

GET /todos

Get all todos

GET /todos/:id

Get specific todo

POST /todos

Post new todo

- Provide text String property on the object.

DELETE /todos/:id

Delete specific todo

PATCH /todos/:id

Update specific todo

- Provide text String property, and/or completed Boolean property.
